# Sveltekit-Breeze

Authentication Starterkit for Sveltekit. Inspired By Laravel Breeze

## Usage

Make your repository using this template repository

## Stack
- Typescript
- Lucia Auth
- Prisma ORM
- Shadncn-svelte
- Tailwind CSS
